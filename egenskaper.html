Color:

Color representerar färgläggning för text helt enkelt till exempel om jag väljer att färga om Hello World! till rött. Se nedan hur koden kan se ut
<span style="color: red">Hello World!</span>

Font:

  • Font-Family olika teckensnitt kan se ut som nedanstående exempel:

    Serif teckensnitt

    Cursive teckensnitt

    Sans-serif teckensnitt

  • Font-Style anger text typsnitt som till exempel:

    Normal style

    Italic style

    Oblique style

  • Font-Size anger typsnitt storlek till exempel:

    font-size 10px

    font-size 20px

    font-size 25px

Text:

  • Text-align anger den horisontella inriktningen av text i ett element till exempel:

    Center text

    Left text

    Right text

    Justify text


    Ovanstående text-align style koden ser ut så här:
    <style="Text-align: center"> Center text
    <style="Text-align: left"> Left text
    <style="Text-align: right"> Right text
    <style="Text-align: justify"> Justify text
  • Text-decoration egenskapen anger dekoration som läggs till text. Till exempel:

    Overline

    Line-through

    Underline

    Underline Overline


    Ovanstående text-decoration style koden ser ut så här:
    <style="Text-decoration:overline">Overline
    <style="Text-decoration:line-through">Line-through
    <style="Text-decoration:underline">Underline
    <style="Text-decoration:underline overline">Underline Overline

Class och ID

I CSS används class och id för att identifiera olika HTML-element. Den största fördelen med att ställa in klass eller ID är att du kan presentera samma HTML-element på olika sätt, beroende på dess class eller ID.

  • Class
    Class selectorn väljer element med en specifik class attribut. Det matchar alla HTML-element baserat på innehållet i deras class attribut. Punkt tillsammans med class namnet används för att välja önskad class.
  • ID
    För att elementet ska kunna väljas måste dess ID-attribut exakt matcha värdet som anges i väljaren. # symbol och id av HTML-element namn används för att välja det önskade elementet.

Word-spacing:

Med den här egenskapen kan man öka eller minska det vita utrymmet mellan ord. Se nedan för exempel:

Lorem ipsum dolor sit amet, consectetur adipiscing elit

Ovanstående word-spacings kod ser ur som nedan:
<p style="word-spacing: 20px">Lorem ipsum dolor sit amet, consectetur adipiscing elit</p>

Background:

  • Background-color

    Background-color ändrar elementets bakgrundsfärg. Alla element har standard och transparent bakgrund. Färger kan specificeras efter smak och behov.

    Exempel:

    Denna bakgrund har aqua färg

    Denna bakgrund har aqua färg

    Ovanstående exempel ser ut så här:
    <div style="background-color: aqua">
  • Background-image

    Background-image ändrar elementets bakgrundsbild. Alla element har standard och transparent bakgrund. Bakgrundsbilder kan specificeras efter smak och behov.

    Exempel:

    Här har jag infogat en bakgrundsbild

    Här har jag infogat en bakgrundsbild

    Ovanstående exempel ser ut så här:
    <div style=" background-image: url('http://images.pexels.com/photos/1591447/pexels-photo-1591447.jpeg?auto=compress&cs=tinysrgb&dpr=1&w=500'); " >
