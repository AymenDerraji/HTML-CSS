Extern CSS

Extern CSS betyder att du använder dig av en separat CSS-fil som innehåller dina deklarationer. Webbsidan länkas sedan ihop med CSS-filen, även kallad för stilmall, och får på så sätt information om hur varje element eller tagg på webbsidan ska visas. Med extern CSS kan du göra stora förändringar i layouten/designen på hela din webbplats utan att du behöver ändra mer än någon enstaka rad i stilmallen.
Ett exempel på just denna CSS-inhämtning är:

<link rel="stylesheet" href="style.css"/>

Intern CSS

Intern CSS är nästan som extern CSS men skillnaden är att det inte finns någon extern stilmall. CSS-deklarationerna läggs istället in mellan <style> elementet i HTML filen. Exempel på det är:

<style type="text/css">
p {
font-family: sans-serif;
font-size: 14px;
}
h1 {
font-size: 36px;}
</style>
